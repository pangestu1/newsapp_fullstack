RESTful API Backend untuk Aplikasi Berita dengan Node.js & MySQL

Base URL: http://localhost:5000/api

üéØ Overview

NewsApp adalah RESTful API backend untuk aplikasi berita dengan sistem role-based authentication. API ini mendukung operasi CRUD untuk berita, komentar, dan manajemen user.

Role Permissions
Admin Full access ke semua endpoint
Penulis Buat, edit, hapus berita (milik sendiri) + komentar
Pembaca Baca berita dan buat komentar

üîê Authentication

POST PUBLIC
/auth/register
Mendaftarkan user baru ke sistem.

Headers:

Content-Type: application/json

Body:

{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "password123",
  "role": "pembaca"
}

Parameters:

Parameter Type Required Description
name string ‚úÖ Nama lengkap user
email string ‚úÖ Email user (unik)
password string ‚úÖ Password (min 6 karakter)
role string ‚ùå admin, penulis, pembaca (default)

Response Success (201):

{
  "message": "User created successfully",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": 1,
    "name": "John Doe",
    "email": "john@example.com",
    "role": "pembaca"
  }
}
POST PUBLIC
/auth/login
Login user dan mendapatkan JWT token.

Body:

{
  "email": "john@example.com",
  "password": "password123"
}

Response Success (200):

{
  "message": "Login successful",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": 1,
    "name": "John Doe",
    "email": "john@example.com",

  }

üì∞ News Management

GET PUBLIC

Mendapatkan daftar semua berita dengan pagination dan search.

Query Parameters:

Parameter Type Required Default Description
page integer ‚ùå 1 Nomor halaman

POST AUTH REQUIRED
/news
Membuat berita baru (hanya Admin dan Penulis).

Headers:

Authorization: Bearer <token>
Content-Type: multipart/form-data

Body (form-data):

Parameter Type Required Description
title string ‚úÖ Judul berita
content string ‚úÖ Konten berita
image file ‚ùå Gambar berita (max 5MB)

üöÄ Quick Start Examples

1. Register User

curl -X POST http://localhost:5000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Alice Johnson",
    "email": "alice@example.com",
    "password": "alice123",
    "role": "penulis"
  }'

2. Create News (with image)

curl -X POST http://localhost:5000/api/news \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "title=Berita Terbaru" \
  -F "content=Konten berita..." \
  -F "image=@/path/to/image.jpg"

3. Get News with Search

curl "http://localhost:5000/api/news?page=1&limit=5&search=teknologi"
